# 快速开始 - 5分钟配置AI功能

## ⚡ 快速配置步骤

### 第1步：获取免费API Key（2分钟）

1. 访问 **[硅基流动](https://siliconflow.cn)** 
2. 手机号或微信登录
3. 右上角头像 → **API密钥** → **创建新令牌**
4. 复制生成的Key（`sk-xxxxxx`）

### 第2步：配置环境变量（1分钟）

打开 `backend/.env` 文件，添加：

```env
SILICONFLOW_API_KEY=sk-你刚才复制的密钥
```

### 第3步：启动项目（2分钟）

```bash
# 启动后端
cd backend
npm run dev

# 新开终端，启动前端
cd frontend
npm run dev
```

## ✅ 测试AI功能

1. 打开浏览器访问 `http://localhost:5173`
2. 登录账号
3. 点击侧边栏 **🌐 全局群聊**
4. 发送几条测试消息
5. 点击 **AI总结最近消息** 按钮
6. 观看AI流式输出效果 ✨
